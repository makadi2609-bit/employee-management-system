# Employee Management System

## 📌 Project Overview
This is a full stack employee management system developed using Java, JDBC, and MySQL. The application allows users to manage employee records efficiently.

## 🚀 Features
- User Registration & Login
- Add Employee
- Update Employee
- Delete Employee
- View Employee List

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Java (JDBC)
- Database: MySQL

## ▶️ How to Run
1. Setup MySQL database
2. Run Java backend
3. Open frontend files in browser
